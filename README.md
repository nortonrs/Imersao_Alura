# Imersao_Alura
O projeto foi baseado em um documento de texto, que é um guia de manutenção em linha de produção de Notebboks, utilizado para reparar defeitos surgidos durante a fabricação. 
Para isso criei uma planilha Excel que contém os tópico de falhas na primeira coluna. Para cada defeito mostrado na primeira coluna a linha correspondente contém as diversas dicas de procedimento para solucionar o problema. 
O usuário recebe uma lista de DEFEITOS (primeira coluna) e escolhe o correspondente. A partir daí é mostrada uma lista das possíveis soluções para o problema. 
O usuário responde com Sim ou Não se o problema foi resolvido. Se Sim, o programa volta ao início. Se Não, pede para mover o equipamento para o EMR (Reparo EletroMecânico).
